# HR-Analytics-Database

HR Analytics – Employee Attrition Analysis

## Objective
Analyze employee data to understand attrition patterns and identify factors impacting
employee retention across departments.

## Tools Used
- SQL (MySQL)
- Data Cleaning & Aggregation

## Dataset
Source: Kaggle – HR Analytics Dataset  
Includes employee demographics, salary, experience, and attrition status.

## Key Analysis Performed
- Overall and department-wise attrition rate
- Salary vs attrition analysis
- Experience and tenure impact on attrition
- Gender and role-based workforce distribution

## SQL Concepts Used
- CASE WHEN statements
- Subqueries
- Aggregations
- Conditional logic

## Key Insights
- Attrition is highest among employees with low tenure
- Certain job roles show consistently high attrition
- Salary growth is inversely related to attrition rate

## Business Recommendations
- Implement retention programs for early-tenure employees
- Review compensation structure for high-attrition roles
- Improve career growth opportunities
